Tridion-powertools

Tridion PowerTools adds additional functionality to the SDL Tridion user interface, that are particularly useful to developers and power users.

View on GitHub Download .zip Download .tar.gz

Code repository and requirements for the Tridion 2011 (and 2013) Power Tools.

Updates

(11-July 2016) No longer under active development. However, most of the tools are available as individual Alchemy plugins. Visit alchemywebstore.com to find out more.

(30-March 2015) We're now on GitHub (https://github.com/TridionCommunity/tridion-powertools). Please only update this Google Code repository to remove or move information to the GitHub.

(13-March 2015) We're moving to GitHub in light of Google Code's "deprecation" status. We'll post updates here and on SDL Community.

(7-August 2014) Thanks to the good people at JetBrains for re-approving another year for a Open Source ReSharper License. Contact one of the PowerTools team if you didn't get the license update.
